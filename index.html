Conexión Supabase

Pega la URL y la Anon Key de tu proyecto Supabase. Los datos se sincronizarán en la nube y podrás acceder desde cualquier dispositivo.

Instrucciones Supabase

1. Entra a supabase.com → tu proyecto
2. Ve a Settings → API
3. Copia la Project URL y la anon public key
4. Pégalas arriba y presiona "Conectar"

Ejecuta este SQL en el SQL Editor de Supabase para crear las tablas:

create table routes ( id uuid primary key default gen_random_uuid(), origen text, destino text, valor numeric, created_at timestamptz default now() ); create table receipts ( id uuid primary key default gen_random_uuid(), num integer, vehiculo text, fecha date, chofer text, area text, solicita text, empresa text, costo text, desde text, hasta text, hinicio text, espera text, fin text, detalle text, total numeric, created_at timestamptz default now() ); alter table routes enable row level security; alter table receipts enable row level security; create policy "allow all" on routes for all using (true); create policy "allow all" on receipts for all using (true);
